# Flappy Bird Version 0.1.0

## Português
Código pra criar um mapa interativo sobre locais com maior índice de vendas com e-commerce em todo o Brasil  
Para esse projeto foi necessário instalar no meu vscode:  
nbformat = define como as células (código, markdown, etc.) são organizadas e salvas no formato .ipynb.   
ipykernel = conecta o código Python ao frontend do Jupyter, seja no browser ou no VS Code.  
plotly = biblioteca de gráficos interativos que pode ser usada para criar visualizações dinâmicas, como gráficos 3D, mapas interativos, e gráficos de série temporal.

## English
Code to create an interactive map about locations with the highest e-commerce sales rates throughout Brazil  
For this project it was necessary to install in my vscode:  
nbformat = defines how cells (code, markdown, etc.) are organized and saved in the .ipynb format.   
ipykernel = connects Python code to the Jupyter frontend, whether in the browser or in VS Code.  
plotly = interactive graphing library that can be used to create dynamic visualizations such as 3D graphs, interactive maps, and time series graphs.
